# docs/public — the public-value layer (PV-00)

This directory is the **generated** public face of `puckworks`: traceable claim
cards derived from the scientific harness, one per public story. It is the concrete
first build of the public-value track (`docs/PUBLIC_VALUE.md`, tasks PV-00–PV-18);
`docs/SPRINTS.md` "Public-value track (PV)" tracks status.

## What is here

- `generated/claims.json` · `generated/claims.csv` · `generated/claims.md` —
  produced by `python -m puckworks.public export`. **Do not hand-edit** — they are
  regenerated from the claim registry.

The claim registry itself is code, not prose:
`puckworks/public/` (`schema.py`, `claims.py`, `export.py`) — a **presentation-only**
package, NOT a registered physics component.

## The integrity guarantee (why this exists)

Every public number is **generated by a named function** (a `Producer` pointing at a
harness/model function), never hand-typed. Each claim also carries, structurally:

- a **unit** on every number;
- an **evidence-strength** label copied UNCHANGED from the scientific analysis
  (`measured/independent` · `post-fit reconstruction` · `verification` ·
  `qualitative` · `reference` · `negative validation`);
- an **observation/reconstruction/prediction/simulation badge**;
- the **source datasets** (which must exist in `puckworks/data/MANIFEST.csv`);
- a validity range, primary caveat (scope sentence), practical implication, and a
  one-line reproduction command;
- the **source commit** it was generated at.

`tests/test_public_claims.py` fails if any of these is missing, if a number is
hard-coded (no Producer), if a grinder-dial comparison lacks a non-portability
warning, or if a stored snapshot has drifted from the live harness result (a revised
scientific result must invalidate its stale card). These are fast structural checks;
they are **not** physics gates and are **not** in `run_all_gates()`.

The public track **consumes** scientific results; it never relaxes their strength
labels (PUBLIC_VALUE.md §3). Reference-strength and labelled-proxy results stay
labelled in public form.

## Regenerate

```bash
python -m puckworks.public export --out docs/public/generated        # fast claims
python -m puckworks.public export --out docs/public/generated --slow # + slow Producers
```
