# docs/archive — superseded documents (retained for provenance)

Everything under `docs/archive/` is **superseded and historical**. It is kept so decisions and
review trails remain auditable, but it is NOT the current source of truth. For the current
documents see [`docs/CURRENT.md`](../CURRENT.md).

## paper_a_reviews/

Successive external detailed-review iterations of Paper A (`PAPER_A_DETAILED_REVIEW` →
`_DRAFT_detailed_review` → `_THIRD_` → `_FOURTH_` → `_UPDATED_`). Each round's findings were
actioned in the manuscript and its producer graph; the reviews are retained as the provenance
of those changes. The **current** manuscript is `docs/PAPER_A_DRAFT.md` and its claims are
verified by `python -m puckworks.paper_a.build verify`.
