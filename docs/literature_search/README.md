# docs/literature_search — Paper A M9 related-work

A **documented scoping search** for Paper A's related work (identifiability /
inverse-problem methods + the coffee-extraction lineage). Seeded from
`docs/PAPER_A_M9_external_dataset_and_decisions_handoff.md` §1, whose reference DOIs
are treated as verified.

| file | status |
|---|---|
| `SEARCH_PROTOCOL.md` | the scoping-search protocol (questions, sources, strings, screening, submission gate) |
| `references.bib` | the verified core references (24 methods + 23 coffee), DOIs from the handoff |
| `EVIDENCE_MATRIX.csv` | one row per screened record — **header only**; populated when the databases are queried at submission |
| `SEARCH_LOG.csv`, `SCREENING.csv` | created at submission (database dates/queries/screening) |

**Important:** this is a scoping search, not a PRISMA-complete systematic review.
The manuscript (§9) therefore says *"to our knowledge, following a documented
scoping search,"* and does **not** assert categorical priority. The full
Scopus/Web-of-Science query is executed and archived at submission (the M9
submission gate in `SEARCH_PROTOCOL.md` lists the closing conditions), which is a
PI/human action — not done here. The insertion-ready related-work section is already
in `docs/PAPER_A_DRAFT.md` §9 and cites only entries in `references.bib`.
