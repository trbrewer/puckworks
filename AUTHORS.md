# Puckworks software authors

Puckworks is co-developed by:

- **Tim Brewer** (`@trbrewer`) — co-developer and software author
- **Peter Vonk** — co-developer and software author

## What this list is (and is not)

- **Software authorship** (this file, `CITATION.cff`, and the `pyproject.toml` `authors` field)
  records the people who co-develop Puckworks the software. Software authorship does **not**
  automatically assign authorship on any specific technical manuscript.
- **Paper / manuscript bylines** are maintained separately, per each paper's actual contribution
  and the authors' explicit approval. They are not derived from this list.
- **Git commit history** contains automated-tool co-author trailers and historical identity records
  (see `.mailmap`) that are **not** the software-author list. In particular, an erroneous historical
  commit identity has been canonicalized to Tim Brewer via `.mailmap` (Git history is not rewritten).

To cite the software, use the metadata in [`CITATION.cff`](CITATION.cff).
