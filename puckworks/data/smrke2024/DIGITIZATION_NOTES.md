# Digitization notes — s41598-024-55831 figure set

Method (all figures): axis calibration by linear fit to detected tick-mark pixel positions (log10 fit for the Fig4 particle-size axis; max calibration residual 7.6e-4 decades, all linear fits exact to sub-pixel). Marker extraction by strict per-series colour masks and connected-component centroids; overlapping markers split by k-means with k from blob area / median single-marker area, cross-checked against disk-template matching and visual zoom inspection. Fig6/Fig7 text-glyph markers identified by Tesseract OCR backed by template correlation against reference glyphs from the same figure; Fig6 labels additionally validated by cross-panel consensus (the 15 yield positions repeat identically in all four panels). QA: every dataset re-plotted over the source image (see qa_overlays/).

## File map
- Fig2: point series by marker colour (black / red; legend not present in the figure, so colour is the defensible label). 15 points.
- Fig3: 46 extractions across 4 fines levels. `markers_in_blob` and `note` columns record every overlap split. Evidence-strength caveats: the 3-marker splits at ~9 s (no-fines) and ~12 s (1 g) were confirmed by zoomed visual inspection but centre positions within those clusters carry ±0.3 s / ±0.05 %-pt uncertainty; one no-fines marker at 17.3 s is partially occluded by a 2 g marker.
